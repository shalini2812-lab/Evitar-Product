# Evitar BFSI AI Workforce Demo

## Overview

This project is an interactive product demo showcasing Evitar's Agentic AI platform for BFSI organizations across Insurance, Housing Finance, NBFCs, and Cards businesses.

The demo helps prospects explore how AI agents can improve customer acquisition, servicing, renewals, field productivity, quality monitoring, and operational visibility through a unified AI workforce.

## Key Features

* Outcome-led solution discovery
* Industry-specific views (Insurance, HFC/NBFC, Cards)
* AI Voice, WhatsApp, Form Assist, Field Buddy, Quality & Training, and Command Center agents
* Shared customer context across channels
* Agent voice and behavior customization
* Real-world BFSI customer journey demonstrations
* Command Center analytics and recommendations
* Governance and compliance controls

## Core Value Proposition

Evitar enables multiple AI agents to operate as a single workforce. Customer context is shared across channels, ensuring seamless experiences across voice, WhatsApp, web, field, and human-assisted interactions without customers repeating information.

## Intended Use

This demo is designed for:

* BFSI CXOs
* Business Heads
* Operations Leaders
* Distribution Teams
* Digital Transformation and AI Stakeholders

## Status

Interactive product demonstration and solution showcase. Built to support customer conversations, solution discovery, and live demos for BFSI prospects.
